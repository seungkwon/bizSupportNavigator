# 기업지원 종합 사이트 
## 프로젝트의 목적

1. 기업의 사업계획서 정보 (JSON)을 이용하여 기업에게 맞는 정책 지원을 매칭해줌.
2.  기업 사업계획서 정보에서 기업의 데모그래픽은 추출하여 데이터베이스에 저장함. (이 기능은 이미 구현되었다고 생각)
3. 정책 사업에 대한 정보는 이미 API를 통하여 수집하여 데이텁제이스에 JSON 형태로 저장 되어있음 (이 기능은 이미 구현 되었다고 생각)
4. 정책 자금의 상세 내용은 첨부 파일로 있으며 이 파일을 파싱하여 기업과 매칭 시키려고 함. 
5. 정책의 메타 정보로 필터링을 하며 (기업규모, 지역, 지원 기간 등) 정책 자금 첨부파일의 내용을  RAG 데이터 및 Graph RAG를 이용하여 
   신청 자격 요청, 신청 제외 요건등을 LLM의 추론 기능 (LangGrpah + RAG + Graph RAG)를 이용하여 구축하여 
   상세한 매칭을 구축하려고 함. 
6. 프론트는 조회 등의 화면과 더불어 채팅 화면을 통하여 사용자에게 질문을 이어  갈 수 있도록 (즉 불충한 정보의 경우) 하며 
   사용자가 직접 입력하기 보다는 선택할 수 있도록  UI를 제공하고자 함. 

## 기술 스택
 1. 프론트엔드: React + 웹소켓 (채팅에서 서버에서 푸시가 필요할 경우 처리)
 2. 서버 : Fast API(RestAAPI + WebSocket) Vector DB + LangGraph + RAG + Graph RAG
